== Description ==
Plugin to translate your website through machine translation. Elhuyar's automatic translator is based on artificial intelligence and neural networks and knows 6 languages. Makes translations between these six languages: Basque, Spanish, French, English, Catalan and Galician. This is a "fork" of the popular WPML multilingual (https://wpml.org/).

== Frequently Asked Questions ==
= Do I need to install the WPML plugin? =
No, if you install this plugin, you don't need to install WPML plugin.

= If I already have the WPML plugin installed, do I have to uninstall it? =
It is not necessary. But I would recommend you to uninstall it, otherwise you'll have to translate the posts twice.

= Is it a free plugin? =
The plugin itself is free, but in order to use the machine translation, You will have to hire a professional fee. More information: https://elia.eus/inicio#prices
